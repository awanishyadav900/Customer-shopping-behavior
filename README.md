# Customer-shopping-behavior



📊 Customer Shopping Behavior Analysis
📌 Project Overview

This project focuses on analyzing customer shopping behavior using PostgreSQL and Data Science tools.
The goal is to extract meaningful insights from customer data such as purchasing patterns, revenue trends, and customer segmentation.

# 🛠️ Tools & Technologies Used

1. PostgreSQL (SQL), Python, NumPy, Pandas, Matplotlib,  Seaborn

Seaborn

# 📂 Project Workflow

 1. Data Storage & Querying

Created and managed database using PostgreSQL

Performed data extraction using SQL queries

2. Data Processing

Loaded data into Python using Pandas

Cleaned and prepared dataset for analysis

3. Data Analysis

Performed exploratory data analysis (EDA)

Used NumPy and Pandas for calculations

4. Data Visualization

Created graphs using Matplotlib and Seaborn

Visualized trends and patterns

# 📊 Key Analysis Performed

. Revenue comparison by gender

. Impact of discounts on purchase behavior

. Top-performing products based on sales and ratings

. Shipping type comparison (Standard vs Express)

. Customer segmentation (New, Returning, Loyal)

. Subscription vs non-subscription analysis

# 📈 Key Insights

. Identified customer spending patterns

. Analyzed effect of discounts on sales

. Found top-selling products and categories

. Segmented customers based on purchasing behavior
